What is STEVE?

STEVE, or the The interstellar agency of Sightseeing and Travel in the EVE Universe, is a fictional agency in the EVE Online universe with the goal to let people experience the beauty, culture and history of the EVE Universe.

What is this website?

The STEVE website is an experiment by Ionaru to try and build a frontend-only Angular application with ESI interaction.

How does it work?

Most of the website is build as a standard Angular application, but with an authentication system using the "OAuth 2.0 flow for native applications". By using the browser's sessionStorage we can navigate away from the website and come back to authenticate the user without the need for a backend. The full code is available on GitHub.
